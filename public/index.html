About

This site was made for fun by me, Matt Bell (mappum), and is not officially affiliated with Y Combinator or Hacker News.

The development stack I used includes Node.js, Express, Socket.IO, Mongoose/MongoDB, Backbone, Bootstrap, and phantomjs. The node hosting is provided by Nodejitsu, and I am hosting the scraper/DB server on a VPS hosted by VPSNode.

There are still quite a few things I would like to add to the site, inluding:

  • Interactivity (logging in, upvoting, commenting, submitting)
  • Viewing comment threads
  • Ability to view all content, not just top 60 posts

If you want to contribute (excuse my messy code), you can fork the code on Github. Also, it would be awesome if you followed me on twitter: Follow @mappum
